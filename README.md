# Projecto HTML- CSS (Website loja de plantas aquáticas)

Este é o meu primeiro website, criado para praticar os conceitos base de HTML e CSS, como a estruturação da página, uso de forms, organização do layout com Box Model e Flexbox, trabalho com imagens, tipografia, cores, espaçamentos e outras propriedades fundamentais. 
O objetivo foi ganhar familiaridade com a construção de páginas web desde o zero e compreender melhor como o HTML e o CSS funcionam em conjunto.
